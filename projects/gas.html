introduction

Genomics Annotation Service (GAS) was developed as the capstone project for my Cloud Computing class at the University of Chicago. As a result of ten weeks of dedicated effort, it shows both an expansive scale and an intricate design. While a basic frontend framework was initially provided, the majority of backend functionalities were independently developed by me. The primary objective of this project was to architect an AWS-based software as a service (SaaS) application that offers genomics annotation services to users.
