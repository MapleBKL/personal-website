introduction

My responsive personal website is my first larger-scale frontend project. I started constructing my personal website after I had learned basic HTML, CSS, and JavaScript. Originally, I was only planning to build a more detailed resume, presenting some project details that could not fit into my one-page resume. However, as I started writing it, I realized it was an excellent opportunity to sharpen my fundamental frontend skills. As a result, I now have a far more polished personal website, and a lot more frontend experience!

My personal website consists of two parts:

  • A homepage presenting basic background information.
  • A dedicated page for each personal project.

The homepage has many animations and design elements, since I want my homepage to be visually attractive. On the other hand, The dedicated project pages have a much simpler design, because I don't want to distract my visitors away from the content.

For responsive design, I started with the desktop version and worked my way down to mobile screens.

There isn't much to write about for this project, since you are interacting with it directly!

lessons learned

What I learned through this project:

  • How to orchestrate CSS animations.
  • How to make a website more accessible.
  • Common design elements and implementation techniques.
  • Good practices in frontend development.
  • Responsive design.
  • As a bonus: how to better use Git.
